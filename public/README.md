# How to Avoid Addiction (V1)

A free, kid-friendly educational web app with **daily lessons**, **mini-quizzes**, and **mini-games**—with **no ads** and **no premium purchases**.

## What’s inside V1
- Home screen + navigation
- 3 starter lessons (easy to expand)
- Quiz completion required to mark a lesson done
- Streak tracking (saved on device)
- Mini-game: “Choice Quest”
- Calm tool: “Breathing Buddy”
- Progress page (completed lessons + high score)

## Tech
- HTML, CSS, JavaScript (no frameworks)
- Runs locally or on any static host (Cloudflare Pages, GitHub Pages, etc.)

## Run it
Just open `index.html` in a browser.

## Notes
Progress is stored using `localStorage` on the current device.
